# summa - Controla tus finanzas

Una aplicación web moderna para automatizar el registro y monitoreo de gastos personales.

## 🚀 Características

- **Automatización completa**: Conecta tu correo para detectar transacciones automáticamente
- **IA integrada**: Clasificación inteligente de gastos usando inteligencia artificial
- **Interfaz intuitiva**: Validación de gastos con gestos simples
- **Seguimiento de presupuesto**: Monitoreo claro y organizado de metas financieras
- **Diseño moderno**: Interfaz elegante con gradientes y efectos visuales

## 📱 Páginas incluidas

- **index.html**: Página principal con landing page y onboarding
- **register.html**: Formulario de registro de usuarios
- **budget-setup.html**: Configuración de presupuesto
- **listo.html**: Página de confirmación

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3 con Tailwind CSS
- JavaScript vanilla
- Fuentes Inter de Google Fonts
- Gradientes y efectos visuales modernos

## 🎨 Diseño

- Paleta de colores: Rosa (#D64A8D) a Púrpura (#8E44AD)
- Tipografía: Inter (400, 500, 700, 800)
- Efectos: Backdrop blur, sombras, gradientes
- Responsive design para todos los dispositivos

## 🚀 Cómo usar

1. Clona este repositorio
2. Abre `index.html` en tu navegador
3. Navega por las diferentes páginas para explorar la funcionalidad
